# Ualbany Campus Service Ratings

Website created by students for students in UA so that students can rate and review campus services. Our goal is to allow freshman and other students learn about services around campus that they didn't know previously existed.

## Features
- Browses services with search and category filters
- View average rating and reviews for each service
- Leave a star rating out of 5 stars and a quick review.

## Requirements
1. Service Directory
  - Displays list of UA campus services
  - Shows the name, category, description and rating of each service
2. Search
  - Text search will filter services by name
  - Category dropdown filters by category
3. Service details
  - Clicking a campus service will open a details page with a description, location and hours with the rating and reviews as well
4. Rating/Reviews
  - Users can submit a rating from 1-5 stars
  - Users can submit a comment for the service
  - New reviews could appear in the detail view & update the list averages


## How to start
1. Clone/download the repo
2. Open __ in your browser (double click it)
3. Click a service card to open details'
4. Leave rating and a comment
